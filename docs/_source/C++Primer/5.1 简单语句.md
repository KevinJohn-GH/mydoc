# 5.1简单语句

一个表达式末尾加上分好就变成了**表达式语句（expression statement）**。表达式语句的作用是执行表达式并丢弃掉求值结果。

#### 空语句

**空语句（null statement）**

```c++
;	//使用空语句时应该加上注释，表明有意省略
```

#### 复合语句（块）

**复合语句（compound statement）**是指用花括号括起来的（可能为空的）语句和声明的序列，复合语句也被称作**块**（block）。一个块就是一个作用域，在块中引用的名字只能在块内部以及嵌套在块中的子块访问。

> 块不以分号作为结束。

空块是指内部没有任何语句的一对花括号，等价于空语句。
